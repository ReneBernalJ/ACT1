## What is Markdown?
Markdown is a lightweight markup language used to format text easily.
It is commonly used to create README files and documentation on GitHub.
Markdown allows the use of headings, lists, links, images, and code blocks.

## Markdown Tagging Options
- Headings (#, ##, ###)
- Bold text (**bold**)
- Italic text (*italic*)
- Lists (ordered and unordered)
- Links and images
- Code blocks

## Git Commands Used
- `git status`: checks the status of the local repository.
- `git add README.md`: adds the file to staging area.
- `git commit -m "message"`: creates a commit with a description.
- `git push`: uploads local changes to the remote repository.
- `git branch`: creates and lists branches.
- `git checkout`: switches between branches.
- `git reset`: rolls back to a specific commit.
